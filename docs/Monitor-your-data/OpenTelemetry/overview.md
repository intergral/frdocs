# OpenTelemetry

OpenTelemetry (OTel) is an industry-standard, open-source observability framework for collecting metrics, traces, and logs. FusionReactor supports receiving OTel data and shipping it to your observability backend via the Observability Agent, the OpenTelemetry Collector, or Grafana Alloy.
