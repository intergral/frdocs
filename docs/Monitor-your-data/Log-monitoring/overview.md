# Overview

Log monitoring in FusionReactor allows you to ingest, visualize, and alert on logs from any location alongside your existing observability data, with automatic shipping from the FusionReactor agent and support for external log shippers.

!!! info "This feature is part of OpsPilot"
    This functionality is available in **OpsPilot**, our cloud-native observability platform.
    [View the OpsPilot documentation](https://docs.opspilot.com/Monitor-your-data/Log-monitoring/overview) or [start a free trial](https://app.opspilot.com).
