# MCP: the Model Context Protocol

The Model Context Protocol (MCP) is a standardized interface that allows AI models to access real-time external data; FusionReactor Cloud provides an MCP integration based on the Grafana MCP project, giving AI tools read-only access to your metrics and logs.
