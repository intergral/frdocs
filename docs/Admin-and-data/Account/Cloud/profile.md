# User Profile

The User Profile page lets you manage your personal account settings, including your display name, profile photo, email addresses, and connected third-party accounts used for single sign-on.

!!! info "This feature is part of OpsPilot"
    This functionality is available in **OpsPilot**, our cloud-native observability platform.
