# FusionReactor Trial | Set up

FusionReactor is an on-premise APM solution giving you deep visibility into your Java and CFML applications. If you also need cloud-based observability, our sister product **OpsPilot** provides full-stack cloud monitoring built on OTel.

## Register for a free trial

| Product | Best for | Get started |
| --- | --- | --- |
| **FusionReactor On-Premise** | Users who need to store data locally or cannot send data to the cloud due to security requirements. | Contact our sales team at **sales@fusion-reactor.com**. Our team will **manually set up your account** and provide your license keys. |
| **OpsPilot** | Users seeking cloud-based data retention, dashboards, alerting, log monitoring, and AI-powered insights. | Visit **[app.opspilot.com](https://app.opspilot.com)** to register and start a free trial. |

## Additional information

- All FusionReactor users have access to the On-Premise user interface.
- If you require local data storage for security reasons, FusionReactor On-Premise is the right choice.
- If you need cloud-based observability and advanced analysis, **OpsPilot** is our recommended solution.

Choose the right option for you and start optimizing your monitoring experience today!
