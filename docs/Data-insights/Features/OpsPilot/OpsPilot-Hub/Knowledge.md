# Knowledge

The Knowledge page lets you build a context library for the OpsPilot AI assistant by adding snippets and files about your technology stack, services, and organisation, helping OpsPilot deliver more accurate and relevant responses.

!!! info "This feature is part of OpsPilot"
    This functionality is available in **OpsPilot**, our cloud-native observability platform.
    [View the OpsPilot documentation](https://docs.opspilot.com) or [start a free trial](https://app.opspilot.com).
