# Request Time Graph

**Requests &gt; Time Graph** displays the average web request performance currently stored by FusionReactor.Â In the top right of the
page next to the **Set Home Page** you can switch graphs quickly by clicking the **Activity** button to be switched to the **WebRequest Activity**
graph.
